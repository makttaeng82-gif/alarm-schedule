알람 스케줄표 사용방법

요일별 반복 일정부터 공부·수업·회의·운동 타이머까지, 메인 화면에서 사용하는 방법을 한 번에 안내합니다.

메인 화면에서 먼저 확인할 영역

번호를 따라가면 일정 등록부터 타이머 실행까지의 흐름을 쉽게 이해할 수 있습니다.

일정 추가 순서

  1. 1

    일정명을 입력합니다

    회의, 수업, 운동처럼 나중에 알아보기 쉬운 이름을 입력합니다.

  2. 2

    반복할 요일을 선택합니다

    월요일부터 일요일까지 원하는 요일을 고릅니다. 오늘 버튼을 누르면 현재 요일이 빠르게 선택됩니다.

  3. 3

    시작·종료 시간을 정합니다

    시작 시간을 선택하면 종료 시간은 한 시간 뒤로 자동 설정됩니다. 필요하면 종료 시간을 직접 바꿉니다.

  4. 4

    알람 조건을 설정합니다

    몇 분 전에 알릴지, 소리, 볼륨, 색상과 메모를 설정합니다. 듣기 버튼으로 소리를 미리 확인할 수 있습니다.

  5. 5

    추가 버튼으로 저장합니다

    저장된 일정은 주간표와 알람 목록에 표시됩니다. 같은 요일과 시간의 일정은 중복 등록할 수 없습니다.

빠른 타이머 사용법

오른쪽 사용자 설정 타이머에서 자주 쓰는 시간을 저장한 뒤, 저장된 버튼을 누르면 현재 시각 기준의 알람 일정이 추가됩니다. 25분 집중, 50분 수업, 10분 휴식처럼 반복해서 쓰는 시간을 등록해 두면 편리합니다.

공부·수업

집중 시간, 수업 종료, 쉬는 시간을 빠르게 알립니다.

회의·업무

회의 시작 전 알림과 업무 블록 종료 시간을 관리합니다.

운동·휴식

운동 시간과 휴식 시간을 놓치지 않도록 알림을 설정합니다.

일상 일정

복약, 외출 준비, 정기적인 집안일을 요일별로 반복합니다.

주간표 확인하기

주간표는 월요일부터 일요일까지 등록된 일정을 한눈에 보여줍니다. 일정 카드를 보면 이름, 시작·종료 시간, 실제 알람 시간이 함께 표시됩니다. 반복 일정이 어느 요일에 등록되어 있는지 확인할 때 사용합니다.

알람 목록 관리하기

알람 목록에서는 저장된 모든 일정을 표로 관리합니다. ON/OFF 버튼으로 알람을 잠시 끄고, 연필 버튼으로 수정하며, 휴지통 버튼으로 삭제합니다. 백업 다운로드와 백업 불러오기도 이 영역에서 실행합니다.

저장한 일정 관리

알람 목록에서 ON/OFF 버튼으로 알람만 잠시 끌 수 있고, 연필 버튼으로 수정하거나 휴지통 버튼으로 삭제할 수 있습니다. 다른 기기로 옮길 때는 백업 다운로드와 백업 불러오기를 사용합니다.

주의: 일정과 타이머는 현재 브라우저에 저장됩니다. 브라우저를 닫거나 컴퓨터가 절전 상태이면 알람이 제한될 수 있습니다.
