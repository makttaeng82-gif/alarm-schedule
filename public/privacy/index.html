개인정보처리방침

어떤 정보가 브라우저에 저장되고 외부 서비스가 언제 사용되는지 공개합니다.

시행일: 2026년 7월 15일 · 최종 업데이트: 2026년 7월 15일

핵심 안내: 일정과 타이머는 회원 계정이나 운영자 데이터베이스에 저장되지 않고 사용자의 현재 브라우저에 저장됩니다.

1. 적용 범위

이 방침은 알람 스케줄표 웹사이트와 사이트에서 제공하는 일정·타이머 기능에 적용됩니다. 사용자가 외부 문의 폼 등 다른 서비스로 이동한 뒤의 정보 처리는 해당 서비스의 정책을 따릅니다.

2. 서비스가 처리하는 정보

브라우저에 저장되는 정보

사용자가 입력한 일정명, 요일, 시작·종료 시각, 사전 알림 시간, 소리, 볼륨, 색상, 메모, 예외 날짜와 사유, 공휴일 제외 상태, 빠른 타이머, 테마 설정이 브라우저의 localStorage에 저장됩니다. 이 정보는 알람 기능을 제공하기 위한 용도로만 사용되며 운영자 서버로 자동 전송되지 않습니다.

브라우저 알림 권한

사용자가 브라우저 알림 켜기를 선택하면 브라우저가 알림 권한을 요청합니다. 허용·차단 상태는 브라우저가 관리하며, 서비스는 현재 권한 상태를 화면에 표시하고 허용된 경우 일정명과 알람 시각을 알림으로 보여줍니다.

백업 파일

백업 다운로드 파일은 브라우저에서 생성되어 사용자의 기기에 저장됩니다. 백업 불러오기도 선택한 파일을 브라우저에서 읽어 형식을 확인한 뒤 현재 브라우저 저장 공간에 반영하며, 파일 내용은 운영자 서버로 업로드되지 않습니다.

3. 외부 통신과 제3자 서비스

공휴일 조회

사용자가 공휴일 자동 불러오기를 실행하면 현재 연도가 사이트의 /api/holidays 경로로 전달됩니다. 서버 기능은 해당 연도만 공공데이터포털 특일정보 API에 요청하며 일정명, 메모, 타이머 같은 사용자 저장 정보는 전송하지 않습니다.

Google 문의 폼

사용자가 문의하기를 눌러 외부 Google Forms 페이지에서 내용을 제출하면, 폼에 직접 입력한 정보가 Google과 폼 운영자에게 전달됩니다. 문의 내용은 답변과 오류 확인을 위해 사용합니다. 외부 페이지의 쿠키와 로그 처리는 Google 정책을 따릅니다.

호스팅 요청 정보

사이트 제공과 보안을 위해 호스팅 사업자가 IP 주소, 브라우저 정보, 요청 시각 같은 일반적인 접속 정보를 처리할 수 있습니다. 해당 정보의 처리는 호스팅 사업자의 정책과 설정을 따릅니다.

4. Google AdSense와 쿠키

알람 스케줄표는 Google AdSense 도입을 준비하고 있습니다. 광고가 활성화되면 Google을 포함한 제3자 광고 사업자가 이전 방문 기록을 기반으로 광고를 제공하기 위해 쿠키, 웹 비콘, IP 주소 또는 그 밖의 식별자를 사용할 수 있습니다.

  • Google과 제휴사는 사용자의 이 사이트 또는 다른 사이트 방문 기록을 기반으로 광고를 제공할 수 있습니다.
  • 사용자는 Google 광고 설정에서 맞춤 광고를 관리하거나 사용 중지할 수 있습니다.
  • Google이 파트너 사이트의 정보를 사용하는 방법은 Google 안내에서 확인할 수 있습니다.

5. 보관과 삭제

브라우저 저장 정보는 사용자가 알람 목록의 전체 초기화를 실행하거나 브라우저의 사이트 데이터를 삭제할 때까지 해당 브라우저에 남습니다. 내려받은 백업 파일은 사용자의 기기에 남으므로 파일 관리 기능으로 직접 삭제해야 합니다.

문의 폼에 제출한 내용은 문의 처리와 반복 오류 확인을 위해 필요한 기간 동안 보관할 수 있습니다. 삭제를 원하는 경우 같은 문의 폼으로 삭제 요청을 보낼 수 있습니다.

6. 이용자의 선택과 권리

  • 일정과 타이머를 직접 수정하거나 삭제할 수 있습니다.
  • 전체 초기화 또는 브라우저 사이트 데이터 삭제로 저장 정보를 제거할 수 있습니다.
  • 브라우저 설정에서 알림 권한을 철회할 수 있습니다.
  • 문의 폼으로 제출한 내용의 확인 또는 삭제를 요청할 수 있습니다.
  • 광고가 활성화된 경우 Google 광고 설정에서 맞춤 광고를 관리할 수 있습니다.

7. 방침 변경

기능, 외부 서비스 또는 관련 정책이 변경되면 이 페이지의 내용과 시행일을 갱신합니다. 중요한 변경 사항은 업데이트 내역 또는 메인 화면에서 안내합니다.
