알람 스케줄표 소개

반복 일정과 한 번만 필요한 타이머를 한 화면에서 관리하도록 만든 개인 운영 웹 도구입니다.

최종 업데이트: 2026년 7월 15일

서비스를 만든 목적

알람 스케줄표는 매주 반복되는 수업·회의·운동 일정과 지금부터 몇 분 뒤 필요한 1회성 타이머를 별도 앱 설치 없이 관리하기 위해 만들었습니다. 반복 일정은 주간표와 알람 목록에서 확인하고, 특정 날짜나 선택한 공휴일만 제외할 수 있습니다.

01

회원가입 없이 사용

계정을 만들거나 로그인하지 않고 현재 브라우저에서 바로 일정을 등록합니다.

02

브라우저 안에 저장

일정과 설정은 운영자 계정 서버가 아니라 사용 중인 브라우저 저장 공간에 보관됩니다.

03

직접 백업·복원

JSON 백업 파일을 내려받아 보관하고, 필요할 때 같은 서비스에서 다시 불러올 수 있습니다.

제공하는 기능

요일별 반복 일정

요일, 시작·종료 시각, 사전 알림, 소리, 볼륨, 색상과 메모를 설정합니다.

1회성 타이머

현재 시각을 기준으로 한 번만 울리는 타이머를 만들고 실행 순서대로 확인합니다.

일정 예외

휴가·휴강·출장처럼 알람을 건너뛸 날짜와 사유를 반복 일정별로 저장합니다.

공휴일 선택 제외

현재 연도의 공휴일을 불러온 뒤 필요한 날짜만 선택해 일정에서 제외합니다.

브라우저 알림

사용자가 권한을 허용한 경우 알람 내용을 브라우저 알림으로 함께 표시합니다.

백업과 초기화

일정을 파일로 백업하거나 브라우저에 저장된 일정 데이터를 직접 초기화합니다.

알람이 작동하는 범위

이 서비스는 운영체제의 예약 알람이 아니라 웹페이지가 실행되는 동안 시간을 확인하는 브라우저 도구입니다. 페이지나 브라우저가 종료되거나 기기가 절전 상태가 되면 알람이 지연되거나 누락될 수 있습니다.

중요한 일정에는 별도 알람을 함께 사용하세요.
알람 스케줄표는 운영체제 알람, 재난 알림 또는 안전 관련 경보를 대신하지 않습니다.

운영과 문의

알람 스케줄표는 개인이 개발하고 운영하는 무료 서비스입니다. 오류 제보와 기능 개선 의견은 Google 문의 폼으로 받고 있습니다. 문의 폼을 열기 전에는 서비스가 문의 내용을 수집하지 않습니다.
