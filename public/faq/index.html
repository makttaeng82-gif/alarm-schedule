자주 묻는 질문

알람 스케줄표를 처음 사용할 때 많이 궁금해하는 내용을 정리했습니다.

회원가입이 필요한가요?

필요하지 않습니다. 브라우저에서 바로 사용할 수 있습니다.

일정은 어디에 저장되나요?

현재 사용 중인 브라우저의 저장 공간에 보관됩니다.

요일별 반복 알람을 만들 수 있나요?

가능합니다. 일정에 원하는 요일을 여러 개 선택하면 됩니다.

타이머는 어떻게 만드나요?

사용자 설정 타이머에서 시간과 분을 선택하고 추가 버튼을 누릅니다.

알람과 데이터

브라우저를 닫아도 알람이 울리나요?

보장되지 않습니다. 페이지가 열려 있고 브라우저가 실행 중일 때 가장 안정적입니다.

일정을 다른 기기로 옮길 수 있나요?

알람 목록에서 백업 다운로드를 선택한 뒤, 다른 기기에서 백업 불러오기를 사용하면 됩니다.

다른 기기와 자동 동기화되나요?

되지 않습니다. 이 서비스는 개인용으로 브라우저에 데이터를 저장합니다.

데이터를 잃지 않으려면
브라우저 데이터를 삭제하기 전에 알람 목록에서 백업 파일을 다운로드하세요.

사용 중 궁금한 점이 있나요?

오류나 개선 의견을 문의 폼으로 보내주세요.
