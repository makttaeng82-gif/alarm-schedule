반복 알람과 타이머 자주 묻는 질문

요일별 반복 알람, 공부 시간 알람, 일정 예외와 브라우저 알림에 대해 자주 묻는 내용을 정리했습니다.

처음 사용하는 경우

회원가입이 필요한가요?

필요하지 않습니다. 브라우저에서 바로 사용할 수 있습니다.

요일별 반복 알람을 만들 수 있나요?

가능합니다. 일정 추가 화면에서 월요일부터 일요일 중 원하는 요일을 하나 이상 선택하면 해당 요일에 반복됩니다.

공부 시간 알람으로 사용할 수 있나요?

가능합니다. 과목명, 공부 요일, 시작·종료 시간, 알람 전 시간을 설정하고 색상과 메모로 계획을 구분할 수 있습니다.

알람은 어디에서 등록하나요?

메인 화면의 일정 추가 영역에서 반복 일정을 등록합니다. 당장 한 번만 필요한 알람은 오른쪽 사용자 설정 타이머에서 추가합니다.

반복 일정과 1회성 타이머

반복 일정과 사용자 설정 타이머는 무엇이 다른가요?

반복 일정은 선택한 요일에 계속 실행되며 주간표와 알람 목록에서 관리합니다. 사용자 설정 타이머는 현재 시각 기준으로 한 번만 울리고 주간표에는 표시되지 않습니다.

사용자 설정 타이머도 일정 예외의 영향을 받나요?

받지 않습니다. 사용자 설정 타이머는 일정 예외와 공휴일 제외와 무관하게 작동합니다.

타이머가 울린 뒤에도 목록에 남나요?

알람을 멈추면 1회성 타이머는 자동으로 삭제됩니다. 울리기 전에는 해당 타이머의 휴지통 버튼으로 취소할 수 있습니다.

일정 예외와 공휴일

휴가나 휴강일만 알람을 제외할 수 있나요?

가능합니다. 일정 예외 설정에서 날짜와 사유를 추가하면 반복 일정은 유지하면서 해당 날짜만 제외합니다. 주간표에는 제외 이유가 표시됩니다.

공휴일에는 자동으로 알람이 꺼지나요?

자동으로 모든 일정이 꺼지지는 않습니다. 공휴일을 자동 불러온 뒤 일정 예외 설정에서 선택한 공휴일 제외를 켠 일정에만 적용됩니다.

공휴일마다 제외 여부를 선택할 수 있나요?

가능합니다. 공휴일을 불러온 뒤 각 항목을 색상으로 선택하거나 해제할 수 있습니다. 목록은 현재 연도 기준으로 관리됩니다.

예외 설정을 저장하지 않고 X를 누르면 어떻게 되나요?

변경 내용이 있으면 저장하지 않고 닫을지 확인하는 안내가 표시됩니다. 저장하지 않으면 팝업을 열기 전 설정으로 돌아갑니다.

브라우저 알람과 데이터

브라우저를 닫아도 알람이 울리나요?

보장되지 않습니다. 페이지를 열어 두고 브라우저가 실행 중인 상태가 가장 안정적입니다. 컴퓨터 절전이나 모바일 브라우저 제한으로 지연될 수도 있습니다.

알람 소리가 들리지 않으면 무엇을 확인하나요?

기기 음량, 일정의 알람 사용 상태와 볼륨, 브라우저 알림 권한을 확인하세요. 메인 화면의 듣기와 10초 후 알람 테스트로 미리 확인할 수 있습니다.

일정은 어디에 저장되나요?

현재 사용 중인 브라우저의 저장 공간에 보관됩니다. 서버 계정에 저장되거나 다른 기기와 자동 동기화되지 않습니다.

일정을 다른 기기로 옮길 수 있나요?

알람 목록에서 백업 다운로드를 선택한 뒤 다른 기기에서 백업 불러오기를 사용하면 됩니다.

브라우저 데이터를 삭제하기 전에 해야 할 일이 있나요?

먼저 알람 목록에서 백업 파일을 다운로드하세요. 브라우저 저장 공간이 삭제되면 백업 없이 복구할 수 없습니다.

중요한 일정에는 별도 알람을 함께 사용하세요.
이 서비스는 브라우저에서 실행되는 개인용 도구이며 운영체제의 예약 알람을 대신하지 않습니다.

답을 찾지 못했나요?

오류나 개선 의견을 문의 폼으로 보내주세요.
